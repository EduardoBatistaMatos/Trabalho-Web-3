Atributos do HTML

1. Atributo href

O atributo href é usado no elemento <a> para especificar o URL de destino para o link.

Exemplo de Código:

        
            <a href="https://www.w3schools.com/html/html_attributes.asp">Visite o Exemplo</a>

2. Atributo src

O atributo src é usado no elemento <img> para especificar o URL da imagem.

Exemplo de Código:

        
            <img src="imagem.jpg" alt="Descrição da Imagem">

3. Atributo alt

O atributo alt é usado no elemento <img> para fornecer um texto alternativo para a imagem, caso ela não seja exibida.

Exemplo de Código:
