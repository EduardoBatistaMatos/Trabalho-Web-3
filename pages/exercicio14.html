Listas HTML

Listas não ordenadas (<ul>)

As listas não ordenadas são usadas para agrupar itens relacionados que não precisam seguir uma ordem específica.

Exemplo de código:

<ul>
    <li>Item </li>
    <li>Item </li>
    <li>Item </li>
</ul>

Exemplo na página:

  • Item
  • Item
  • Item

Listas ordenadas (<ol>)

As listas ordenadas são usadas para agrupar itens relacionados que precisam seguir uma ordem específica.

Exemplo de código:

<ol>
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
</ol>

Exemplo na página:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas de descrição (<dl>)

As listas de descrição são usadas para apresentar termos seguidos de suas definições correspondentes.

Exemplo de código:

<dl>
    <dt>Termo 1</dt>
    <dd>Definição 1</dd>
    <dt>Termo 2</dt>
    <dd>Definição 2</dd>
</dl>

Exemplo na página:

HTML
Linguagem de marcação de hipertexto.
CSS
Linguagem de folha de estilo em cascata.
