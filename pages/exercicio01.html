Elementos Básicos do HTML

1. <h1> - <h6>

Os elementos basicos são usados para definir os cabeçalhos das seções. Existem seis níveis de cabeçalho, onde <h1> é o mais importante e <h6> o menos importante.

Exemplo de Código:

        
            <h1>Título Principal</h1>
            <h2>Subtítulo</h2>
            <h3>Subtítulo Menor</h3>

Exemplo Implementado:

Título Principal

Subtítulo

Subtítulo Menor

2. <p>

O elemento <p> define um parágrafo de texto.

Exemplo de Código:

        
            <p>Este é um parágrafo de exemplo.</p>
            <p>Outro parágrafo aqui.</p>

Exemplo Implementado:

Este é um parágrafo de exemplo.

Outro parágrafo aqui.

3. <a>

O elemento <a> define um hiperlink, que pode ser usado para vincular uma página a outra.

Exemplo de Código:

        
            <a href="https://www.example.com">Visite o Exemplo</a>
